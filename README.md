# STM8_IR_TOOLKIT
Just collection of stm8 code, it is quite heavy now for 8KB of space, needs to be rewritten without SPL probably to save more space, this is it for now. For control small tactile joysticky like switch is used. 
Display used is similar to TM1650, just with small writing changes. I used external SPI flash for saving IR durations, this logic can be definitelly repurposed for something else.
